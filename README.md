## 💜 О проекте

**Gem Blitz** — это динамичная аркадная игра в жанре «три в ряд», разработанная на Python с использованием библиотеки Pygame.

Игроку предстоит собирать комбинации одинаковых фишек, зарабатывать очки и достигать максимального результата за ограниченное время.

---

## ✨ Особенности

* 💎 Классический геймплей «Match-3»
* ⚡ Быстрые и плавные анимации
* 🔥 Система комбо
* ⏱ Ограничение по времени
* 🎯 Набор очков и рекорды
* 🎵 Звуковое сопровождение
* 🎨 Яркий и минималистичный интерфейс

---

## 🕹 Геймплей

* Собирайте 3 и более одинаковых элемента в ряд
* Получайте очки за каждую комбинацию
* Делайте комбо для увеличения счёта
* Старайтесь набрать максимум очков до окончания времени

## 🚀 Установка и запуск

```bash
git clone https://github.com/Anyusek/gem-blitz.git
cd gem-blitz
pip install pygame
python main.py
```
